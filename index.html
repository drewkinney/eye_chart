The idea is to take a data block and split into a auto generating eye chart. The type of data is irrelevant (mysql, xml, txt, etc..).

The data would be slit into this pattern: 1 23 456 7890

Each line would include an auto-incremental class assignment in html for css control.

The first version will be php. I would like to collect many different language variants, as well.
